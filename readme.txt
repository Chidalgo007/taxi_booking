<h1>🚖 Simple Taxi Booking System</h1>

An interactive, real-time taxi booking system built with HTML, CSS, JavaScript, and PHP — designed for both customers and administrators.<br/>
🗺️ Integrated with <strong>Google Maps</strong> for address prediction and route display.<br/>
📄 Generates booking references and updates status live.<br/>
🔐 Includes an admin dashboard for booking management.<br/>

<br/>

<img src="https://your-image-url.com/banner.png" alt="Taxi Booking Banner" width="100%" />

<br/><br/>

<h3>📌 Features</h3>

<h4>👤 Client Side</h4>
📍 Address autocomplete using Google Maps<br/>
🗺️ Map displays route from point A to B<br/>
✅ Validates user inputs before booking<br/>
🧾 Generates unique Booking Reference (e.g., <code>BRN12345</code>)<br/>
🔄 Booking status updates automatically when assigned<br/>

<br/>

<h4>🔐 Admin Side</h4>
🔍 Search bookings by reference, name, or phone number<br/>
⏰ View all bookings scheduled in the next 2 hours<br/>
✅ Assign bookings with one click<br/>
📡 Status updates instantly visible on the client side<br/>

<br/>

<h3>🛠️ Tech Stack</h3>

Frontend: HTML5, CSS3, JavaScript<br/>
Backend: PHP (Vanilla)<br/>
Database: MySQL (PDO)<br/>
API: Google Maps JavaScript API (Places + Directions)<br/>

<br/>

<h3>🧠 How It Works</h3>
1️⃣ User fills out a booking form with pickup/dropoff<br/>
2️⃣ Google Maps autocompletes and validates the address<br/>
3️⃣ Booking is saved to MySQL and reference is generated<br/>
4️⃣ Admin sees all bookings due in 2 hours<br/>
5️⃣ Admin can search, assign, and update booking statuses<br/>
6️⃣ Clients get live status updates instantly<br/>
